 How to load data from subtitle (
 `.srt`
 ) files
```
from langchain.document\_loaders import SRTLoader
```
```
loader = SRTLoader("example\_data/Star\_Wars\_The\_Clone\_Wars\_S06E07\_Crisis\_at\_the\_Heart.srt")
```
```
docs = loader.load()
```
```
docs[0].page\_content[:100]
```
