 CoNLL-U
 [#](#conll-u "Permalink to this headline")
=====================================================
 This is an example of how to load a file in
 [CoNLL-U](https://universaldependencies.org/format.html) 
 format. The whole file is treated as one document. The example data (
 `conllu.conllu`
 ) is based on one of the standard UD/CoNLL-U examples.
```
from langchain.document\_loaders import CoNLLULoader
```
```
loader = CoNLLULoader("example\_data/conllu.conllu")
```
```
document = loader.load()
```
