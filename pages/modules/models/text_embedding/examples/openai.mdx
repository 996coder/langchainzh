 Let’s load the OpenAI Embedding class.
```
from langchain.embeddings import OpenAIEmbeddings
```
```
embeddings = OpenAIEmbeddings()
```
```
text = "This is a test document."
```
```
query\_result = embeddings.embed\_query(text)
```
```
doc\_result = embeddings.embed\_documents([text])
```
 Let’s load the OpenAI Embedding class with first generation models (e.g. text-search-ada-doc-001/text-search-ada-query-001). Note: These are not recommended models - see
 [here](https://platform.openai.com/docs/guides/embeddings/what-are-embeddings) 
```
from langchain.embeddings.openai import OpenAIEmbeddings
```
```
embeddings = OpenAIEmbeddings(model\_name="ada")
```
```
text = "This is a test document."
```
```
query\_result = embeddings.embed\_query(text)
```
```
doc\_result = embeddings.embed\_documents([text])
```
